# 🏳️ myCountryAndFlag

Веб-гра: **збери заданий прапор** зі смуг і реальних кольорів прапорів світу. Плюс режим вільного конструктора.

**Грати:** https://anatoliistepaniuk.github.io/myCountryAndFlag/

## Режими

- **🎯 Гра** — програма називає країну, ти складаєш її прапор (напрям смуг + кольори) і тиснеш «Перевірити». Рахунок = вгадано − пропущено.
- **🎨 Конструктор** — вільно твори будь-який прапор (з підписами кольорів за країнами).

## Перед грою

- **Континенти** (Європа / Азія / Африка / Америка) — з лічильником доступних країн.
- **Точний відтінок** — чи потрібен саме оригінальний відтінок, чи будь-який відтінок кольору.
- **Складні прапори** — вмикати чи ні прапори з нерівними смугами (Таїланд, Латвія…).

## Як влаштований матчинг

- Прапор = напрям (вертикальні / горизонтальні смуги) + послідовність кольорів.
- **Товщина смуг не важлива**: сусідні однакові смуги рахуються як одна широка. Тому Таїланд можна зібрати як 5 або 6 смуг, Латвію — як 3 або 5.
- Перевірка окремо оцінює **напрямок**, **кольори** і **розташування**.

## Технічно

- Один статичний файл `index.html` — HTML + CSS + JS, без залежностей; SVG-рендер; GitHub Pages.
- 30 країн; палітра кольорів генерується з бази прапорів.

## Локальний запуск

```bash
python3 -m http.server 8000 --directory .
```

Відкрий http://localhost:8000

# 🏳️ myCountryAndFlag

Веб-гра-конструктор прапорів: на базі реальних кольорів прапорів світу склади власний прапор зі смуг.

**Грати:** https://anatoliistepaniuk.github.io/myCountryAndFlag/

## Як грати

1. Обери **розташування смуг** (суцільний / 2–3 вертикальні / 2–3 горизонтальні).
2. **Торкнись секції** прапора, щоб її обрати.
3. Обери **колір** з палітри реальних відтінків прапорів (згруповані за основним кольором, підписані країною).

## Технічно

- Один статичний файл `index.html` — HTML + CSS + JS, без залежностей.
- Прапор малюється як **SVG**; кожна смуга — окремий прямокутник, який можна обрати дотиком.
- Розгортається як **GitHub Pages**.

## Локальний запуск

```bash
python3 -m http.server 8000 --directory .
```

Відкрий http://localhost:8000

## Плани

- Режим **«Збери заданий прапор»** з перевіркою правильності.
- Екран налаштувань: вибір **континентів** (Європа / Азія / Африка / Америка / весь світ).
- Два рівні складності: **будь-який відтінок** ↔ **точний відтінок** оригіналу.
- Довільна кількість смуг.
